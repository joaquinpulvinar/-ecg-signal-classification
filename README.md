# ECG Signal Classification

This project explores machine learning methods for classifying ECG signals
under noisy measurement conditions.

The goal is to study feature-based classification of physiological time-series
data, with an emphasis on signal robustness and statistical evaluation.

## Project Structure

- data/       : dataset links and metadata (no raw data)
- notebooks/  : exploratory analysis and experiments
- src/        : preprocessing, feature extraction, and models
- results/    : figures and evaluation outputs
